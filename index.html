    Introduction

    Welcome to the Hexiro API! You can use our API to access Hexiro API endpoints.

    Authentication

    Hexiro will use API keys to allow access to the API.

    Hexiro will expect for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: HEXAPIKEY

    You must replace HEXAPIKEY with your personal API key.
    Authorization codes may or may not be used in the future, currently are not required.

    Groups

    Get a Specific Group

    curl "https://hexiro.com/api/v1/group/1"
  -H "Authorization: HEXAPIKEY"

    The above command returns JSON structured like this:

    {
  "data": {
    "id": 1,
    "owner_id": 2,
    "name": "Hexiro",
    "description": "Hello there!",
    "members": 101,
    "shout": null,
    "funds": null,
    "created_at": {
      "date": "2018-12-31 22:03:34.000000",
      "timezone_type": 3,
      "timezone": "UTC"
    }
  }
}

    This endpoint retrieves a specific group.

    HTTP Request

    GET https://hexiro.com/api/v1/group/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the group to retrieve

    Users

    Get a Specific User

    curl "https://hexiro.com/api/v1/user/1"
  -H "Authorization: HEXAPIKEY"

    The above command returns JSON structured like this:

    {
  "data": {
    "id": 1,
    "username": "Hexiro",
    "description": "neeeerds",
    "signature": "",
    "Membership": null,
    "AvatarURL": "https:\/\/s3.us-east-2.amazonaws.com\/hexiro\/t1\/74f879a3c9ce04c1f6e14a69d60fd005.png",
    "created_at": {
      "date": "2018-10-05 02:20:08.000000",
      "timezone_type": 3,
      "timezone": "UTC"
    }
  }
}

    This endpoint retrieves a specific user.

    HTTP Request

    GET https://hexiro.com/api/v1/user/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the user to retrieve

    Errors

    The Hexiro API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The kitten requested is hidden for administrators only.
    404 Not Found -- The specified kitten could not be found.
    405 Method Not Allowed -- You tried to access a kitten with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The kitten requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
